Este é um jogo de CONEXÕES.

Jogadores: máximo de 5 jogadores

Tempo de duração: 1 hora de estudo e 2 horas de jogo.

O objetivo é criar conexões entre os artefatos artísticos (a poética e a prática) e as pedagogias (a teoria) com argumentos que relacionem os artefatos ou pedagogias entre si. Em que se relacionam? que conceitos são comuns? que ideias são similares? que visões compartilham? que estratégias são semelhantes? que condições são análogas? que elementos são comparáveis? que ideias são transferíveis?

Se apresentam cinco artefatos de artistas/professores nos blocos com imagens, cada um com uma ficha técnica, textos e/ou links de leitura e quatro pedagogias nos blocos de texto, cada uma com leituras e/ou vídeos. Este jogo exige um ritual de preparação:

É importante conhecer um pouco sobre cada proposta pedagógica, investigar.

Instruções para o jogo:

Para rolar o dados de 6 caras deve se manter aberto este link  AQUI

1 São duas rondas de jogo, ou mais se for possível. A cada ronda os participantes jogam o dado e segundo o número que tiraram tem prioridade de escolher uma imagem das cinco disponíveis, para investigar. Os participantes vão iniciar sua investigação ali.

2  O primeiro da numeração inicia a ronda. Na sua jogada deve rolar o dado e conectar as informações sobre sua imagem com as ideias da:

Para 1 - Pedagogia do Oprimido

Para 2- Pedagogia espetacular

Para 3- Pedagogia do evento

Para 4- Pedagogia cultural

Para 5- Escolhe

Para 6- Escolhe dois conceitos para conectar

 

Depois de que todos tenham jogado o dado começa o tempo da leitura e da escrita. Poderão se apoiar no material variado de pesquisa que oferece o jogo. Uma vez que concluem as duas rondas os textos são apresentados ao grupo nas ficha de conexões.
